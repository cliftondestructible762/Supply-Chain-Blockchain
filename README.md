## 🔑 Setting Up Your Wallet

This application uses technology that requires a digital wallet to track your interactions. Install the MetaMask extension in your web browser to manage these activities.

1. Visit the Chrome Web Store or your browser's add-on store.
2. Search for "MetaMask".
3. Click "Add to browser".
4. Create a new wallet when prompted.
5. Write down your "Secret Recovery Phrase" and store it in a safe place. Never share this phrase with anyone.
6. The application connects to this wallet once you launch it.

## 🛠️ Frequently Asked Questions

*Why does the application need a wallet?*
The wallet proves your identity when viewing secure records on the blockchain. It keeps your activity private and linked to your account.

*Is my data public?*
Records on the blockchain are transparent, but the system obscures sensitive personal information. You only see the flow of goods through the supply chain.

*What if I forget my wallet password?*
You use the secret recovery phrase to regain access. Without this phrase, you lose access to your account and historical records.

*Does this software run offline?*
No, the application requires an internet connection to sync with the network. If you lose your connection, wait for it to return and refresh the page.

*Can I trust the data?*
Yes, the blockchain prevents unauthorized changes. The history of each product is permanent and verifiable by all stakeholders.
